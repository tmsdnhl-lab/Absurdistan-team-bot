# 🎵 Absurdistan Team Bot

**Automatizovaný Facebook marketing bot pro hudební promotování**

---

## 📋 Funkce

- ✅ **Plánování příspěvků** – připravuj, schvaluj a automaticky publikuj
- 📊 **Monitoring sociálních sítí** – sleduj zmínky a trendy
- 📈 **Analytika** – počty lajků, komentářů, dosahu
- 🎯 **Hledání influencerů** – automatické notifications
- 📅 **Správa termínů** – připomínky na vydání tracků
- 🔍 **Trend monitoring** – sledování hip-hopu trendů
- 📧 **Automatické notifikace** – Discord/email alerts

---

## 🚀 Rychlý start

### Požadavky
- Node.js 14+
- Facebook Developer Account
- Replit (zadarmo hosting)

### Instalace

```bash
git clone https://github.com/tmsdnhl-lab/absurdistan-team-bot.git
cd absurdistan-team-bot
npm install
cp .env.example .env
npm start
```
